CDN Integration

Include the template directly in your HTML:



<link rel="stylesheet" href="https://cdn.templatehub.com/templates/[TEMPLATE-ID]/style.css">


<div id="template-container"></div>


<script src="https://cdn.templatehub.com/templates/[TEMPLATE-ID]/script.js"></script>

NPM Package

Install via npm:

npm install @templatehub/[template-name]

Then import in your project:

import '[template-name]' from '@templatehub/[template-name]';
